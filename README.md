# Todo List Android Application

## Description
This project is a Todo List mobile application developed as a university project using
Android and Kotlin. The application allows users to manage daily tasks efficiently
through a simple and user-friendly interface.

## Project Scope
- Create and manage daily tasks
- Mark tasks as completed
- Remove completed or unwanted tasks
- Provide a clean and intuitive mobile user interface

## Technologies Used
- Kotlin
- Android Studio
- Gradle

## Features
- Add new tasks
- View all tasks in a list
- Mark tasks as completed
- Delete tasks
- Search tasks
- Simple and responsive UI design

## Project Structure
- `app/src/main/java` – Application logic written in Kotlin
- `app/src/main/res` – UI layouts and resources
- `build.gradle.kts` – Project build configuration

## How to Run
1. Install Android Studio
2. Clone or download this repository
3. Open the project in Android Studio
4. Sync Gradle files
5. Run the application on an emulator or physical device
